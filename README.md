# mm_invest
M'M invest  je interaktivni kalkulator ulaganja za usporedbu mirovinskih strategija — 3. mirovinski stup, PEPP i globalnih ETF fondova (VWCE, S&amp;P 500, Nasdaq i dr.). Omogućuje projekciju rasta portfelja, simulaciju kombinirane strategije s realnom vrijednošću i procjenom mjesečne isplate prema 4% pravilu.
